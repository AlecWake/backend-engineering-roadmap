# Python Grade Tracker (Week 3)

This project is part of my backend development roadmap.

## Features

- Add students and grades
- Update student grades
- Delete students
- View all grades
- Data persistence using a text file

## Concepts Practiced

- Python dictionaries
- Functions
- Exception handling (try/except)
- File reading and writing
- Modules and imports
- Virtual environments
- requirements.txt
- .gitignore

## Project Structure

main.py – program logic and menu system  
storage.py – handles saving and loading data  
grade-report.txt – stores student grade data
